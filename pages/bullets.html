# header
you can type --- as bullet text and it will create a divider - a horizontal line.

you can type ---text as bullet text in which "text" can be anything. It will create a divider and display the text on the far left side. You can use this for timeboxing purposes. Adding a bullet text like ---10:00 will create a divider with 10:00 on the left side.
